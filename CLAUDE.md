## Personal Files

- `my_inputs/` — personal paper inputs, organized by project (e.g., `my_inputs/nafld_review/`). Gitignored.
- `.env` — API keys. Gitignored. Source before running.
- `configs/model_config.yaml` — local config, gitignored. Uses new field names: `main_model_name`, `image_gen_model_name`. Stack B (Hybrid Frontier) is the default.

## Model Stacks

`configs/model_config.yaml` documents three stacks. Active default: **Stack B — Hybrid Frontier** (`google/gemini-3.1-pro-preview` + `openai/gpt-5.4-image-2`). Swap by editing the two `defaults:` lines; alternatives are commented in the file.

| Stack | Text | Image | Notes |
|---|---|---|---|
| A | `google/gemini-3.1-pro-preview` | `google/gemini-3-pro-image-preview` | Cheapest; one provider |
| **B (default)** | `google/gemini-3.1-pro-preview` | `openai/gpt-5.4-image-2` | Best scientific reasoning + arena-#1 image |
| C | `openai/gpt-5.5` | `openai/gpt-5.4-image-2` | OpenAI cohesive; priciest text |

Streamlit dropdowns (`demo.py`) expose all stack models as presets; the config default lands at index 0.

## Per-Project Style Guides

The Stylist and Polish agents read style guides from `style_guides/{prefix}_{diagram|plot}_style_guide.md`. The prefix is set via the `PAPERBANANA_STYLE_GUIDE_PREFIX` env var (default: `neurips2025`).

| Project | Prefix | Style guides |
|---|---|---|
| Default / NeurIPS-flavoured scientific figures | `neurips2025` | Auto-generated by `style_guides/generate_category_style_guide.py` from `data/PaperBananaBench/` |
| Correlation to Causation (Substack editorial) | `correlation_to_causation` | Hand-crafted: `correlation_to_causation_diagram_style_guide.md`, `correlation_to_causation_plot_style_guide.md` |

Switch projects:

```bash
# Editorial / C2C
export PAPERBANANA_STYLE_GUIDE_PREFIX=correlation_to_causation
streamlit run demo.py

# Back to default
unset PAPERBANANA_STYLE_GUIDE_PREFIX
```

To add a new project: drop two files (`{prefix}_diagram_style_guide.md`, `{prefix}_plot_style_guide.md`) in `style_guides/` mirroring the section structure of `neurips2025_diagram_style_guide.md`, then export the prefix. Hand-crafting is preferred when you have a clear aesthetic; the auto-generation script (`generate_category_style_guide.py`) is for mining style from a 50+ figure reference corpus.

Code references: `agents/stylist_agent.py:62`, `agents/polish_agent.py:51`.

## Running

Upstream ships two demos. **Default to Streamlit** (`demo.py`) — it's the more mature surface and matches user preference. Use Gradio (`app.py`) only when explicitly mirroring the HF Spaces deployment.

```bash
# Streamlit (default)
source .env && streamlit run demo.py

# Gradio (HF Spaces parity only)
source .env && python app.py
```
